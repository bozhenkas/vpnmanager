Realtime media sync and delivery gateway

Secure WebSocket transport for client synchronization, presence updates and media delivery sessions. Built for low-latency delivery across distributed edge nodes.

Transport

Authenticated WebSocket channels over TLS 1.2/1.3, multiplexed per client session.

Edge routing

Requests are routed to the nearest healthy edge node; unhealthy nodes are removed from rotation automatically.

Sessions

Session tokens are issued out-of-band and are not accepted on this endpoint without a valid Upgrade header.
